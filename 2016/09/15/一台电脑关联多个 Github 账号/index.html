一台电脑关联多个 Github 账号

发表于 2016-09-15 | 分类于 工具使用
字数统计: 219 | 阅读时长 ≈ 1

生成不用的 SSH key

输入命令生成 SSH key：ssh-keygen -t rsa -b 4096 -C "xxxx@gmail.com"
注：一直回车就好

输入命令生成 SSH key：ssh-keygen -t rsa -b 4096 -C "yyyy@foxmail.com"
注：

  • 需要输入路径文件名：/Users/username/.ssh/xxx_rsa
  • 如果不输入就会存在 /Users/username/ 下

把 SSH key 添加到 Github 网站

…：Setting -> SSH Key -> add

添加 config 账号

添加配置文件 config

验证 SSH key

  • ssh -T github.com
  • ssh -T xxx_github.com

具体项目中修改 .git/config 文件

直接使用命令修改：
git remote set-url origin xxx_github.com:xxx/yourProject.git
