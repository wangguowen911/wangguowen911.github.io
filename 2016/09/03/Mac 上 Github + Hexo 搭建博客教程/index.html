Mac 上 Github + Hexo 搭建博客教程

发表于 2016-09-03 | 分类于 博客
字数统计: 985 | 阅读时长 ≈ 4

下面是自己学习搭建 Hexo 博客的一些记录，为了方便后面查阅，也为了让还没搭建过的新手少走一点弯路！

搭建过程，基础配置搭建

1、安装 Node.js 版本管理工具：nvm

注：npm 维基百科（全称Node Package Manager，即node包管理器）

安装命令：curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.0/install.sh | bash

安装完成后根据提示重启 terminal，并执行：(username 更换成你自己的用户名，下面的命令会在 nvm 安装完成后提示给用户)

2、安装 Node.js

  1. 列出 Node 版本列表：nvm ls-remote
  2. 安装 Node.js ：nvm install v9.3.0
  3. 显示当前安装的 Node 版本：node -v
  4. 查看 npm 版本：npm -v
  5. 查看目前安装的所有nodejs版本及当前使用的版本：nvm list

注：node.js 官网上 http://nodejs.org 下载的 pkg 包安装缺少很多东西，最后还是需要使用 PKG Uninstaller 这个软件来卸载安装，所以推荐使用命令行安装。

3、安装 Hexo

  1. 安装 cnpm： npm install -g cnpm
  2. 安装 Hexo：cnpm install -g hexo-cli
  3. 查看 Hexo 版本：hexo -v

4、初始化 Hexo 项目

  1. 在桌面新建 hexoDemo 项目文件夹
    1. cd Desktop
    2. mkdir -p hexoDemo
  2. 初始化 hexo 项目：hexo init（这一步可能会非常慢）

5、启动 Hexo 服务

  1. 启动：hexo server (需要在 hexoDemo 项目目录下执行)
  2. 创建新文章：hexo new "Hello Hexo"

Hexo 主题

Github 上有很多 Hexo 主题可以下载安装，比如 maupassant、next等等。
下面只是介绍 next 主题的安装，其他的类似下面的步骤进行安装。

hexo-theme-next：

  • Theme Learn from 小土刀
  • next 主题 GitHub
  • next 主题基本设置

hexo-theme-next 主题的安装：

  1. cd hexoDemo 目录
  2. 创建目录 mkdir themes/next
  3. 执行安装命令：curl -s https://api.github.com/repos/iissnan/hexo-theme-next/releases/latest | grep tarball_url | cut -d '"' -f 4 | wget -i - -O- | tar -zx -C themes/next --strip-components=1

安装 next 需要先安装 wget

多台电脑管理 Hexo 项目

  1. 在 hexo 项目下初始化 git
  2. gitignore 文件会在 hexo 初始化的时候就自动创建了
  3. 提交项目文件到远程
  4. 其他电脑下载项目文件后执行：npm install，如何就可以运行了
  5. 如何开启 hexo 服务：hexo server -p 8199，使用其他端口，避免被占用
