3.1、Xcode

Xcode 代码块设置

添加操作：选中代码，单击，长按，然后拖动到右下角代码块区域

变量的使用：<#xxxName#>
Example：<#mark#>、<#type#>、<#value#>

快速打开代码块所在文件夹：
open ~/Library/Developer/Xcode/UserData/CodeSnippets

上面这个目录是你添加了自定义代码块之后才会出现

代码块命名规则说明：

  1. vXXX：表示变量(Variable) 的代码块
  2. focmethod：表示方法(OC Function) 的代码块
  3. fswiftmethod：表示方法(Swift Function) 的代码块
  4. mark：表示添加：// MARK: - XX

3.2、Alfred（效率神器，简直太赞了）

  1. 呼出Alfred：alt + 空格
  2. 关机：shutdown，输入 shut 就可以选择了
  3. 重启：restart
  4. 强制退出应用：forcequit

Alfred 搜索框不仅仅局限于打开当前应用，还可以进行网页搜索
after：alt + 空格 输入内容，回车全键盘操作
before：鼠标点击浏览器，等待加载，输入内容，回车需要使用鼠标

3.3、Subline Text

  1. 选择语言方法一：点击右下角进行选择
  2. 选择语言方法二：shift+Ctrl+p 输入：ssr
  3. 把文件拖到 Subline Text App 图标上，就可以打开整个文件夹啦
  4. 显示文件边栏：view ——> side bar ——> Hidden/Show side bar
  5. 列编辑：cmd+shift+L
  6. 格式化代码：
    1. Edit -> Line -> Reindent
    2. 格式化代码：CodeFormatter
  7. 对齐插件：Sublime Alignment、VAlign(基于Sublime Alignment做的)、Code Alignment
  8. 注释插件：docblocker
