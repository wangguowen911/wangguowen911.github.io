开启 Apache 服务

  1. 进入root用户模式下：$ sudo su -
  2. 进 Apache 目录：$ cd /etc/apache2/
  3. Apache 开启：$ apachectl start
  4. Apache 关闭：$ apachectl stop
  5. Apache 重启：$ apachectl restart
  6. 本地访问：http://localhost/ （是否有出現 “It works!”，有就代表 Apache 已經正常運作了）
  7. 配置目录：/Library/WebServer/Documents

Mysql

安装

  1. 方法一：mysql 官网，下载 dmg，一路默认安装
  2. 方法一：使用 HomeBrew 安装 brew install mysql

配置

设置PATH：
export PATH=/usr/local/mysql/bin:$PATH
echo $PATH

安装完成后的提示：

开启 mysql server

启动：

  1. 开启 mysql server：$ mysql.server start
  2. 登录 mysql： $ mysql -u root -p，然后输入登录密码

关闭：

  1. 退出登录 mysql：quit or exit
  2. 关闭 mysql server：$ mysql.server stop

其他：

  1. 设置登录 mysql 密码 SET PASSWORD FOR 'root'@'localhost' = PASSWORD('123456');
  2. 查看 mysql 帮助：quit
  3. 命令行使用 mysql 的时候，是以 ; 为结束符号，输错了可以直接输入 ; 结束当前输入

使用

参考：MySQL之终端（Terminal）管理数据库、数据表、数据的基本操作

数据库操作：

  1. 显示所有数据库：show databases;
  2. 删除某个数据库：drop database xxxDB;

表操作：

  1. 进入某个数据库：use xxxDB;
  2. 查看当前使用的数据库：select database();
  3. 显示当前数据库所有的数据表：show tables;
  4. 建表
  5. 显示表结构：desc xxxTable;
  6. 然后就是增删改查，修改表结构等等

安装部署 PhpMyAdmin

要管理Mysql，如果用命令行比较麻烦，开源的 phpMyAdmin 采用C/S的模式，方便管理。

phpMyAdmin 下载地址，选择 zip 格式文件下载，然后解压

解压后放置下面的目录
完整的目录为：/Library/WebServer/Documents/phpmyadmin/，
那么命令行进入这个目录，输入以下命令：
cp config.sample.inc.php config.inc.php
vim config.inc.php

接着我们就可以在浏览器中输入URL：http://localhost/phpmyadmin/
输入 MySql 的用户名和密码就可以管理 MySql 的数据表了
